# 파일 백업 프로그램 (2019년 봄학기 리눅스 시스템 프로그래밍 과제)

## 개요
Multi-Thread를 이용하여 특정 파일 또는 디렉토리를 주기적으로 백업하는 프로그램입니다. 백업 대상은 리스트에 등록되며, 각 스레드가 독립적으로 백업을 수행합니다.

- **과목**: 리눅스 시스템 프로그래밍
- **학기**: 2019년 봄학기
- **점수**: 96 / 100

## 실행 환경
- OS: Ubuntu 18.04 LTS
- Compiler: gcc 7.5.0
- Language: C

## 주요 기능
- 파일 또는 디렉토리를 백업 리스트에 등록
- 각 항목을 별도 스레드로 주기적으로 백업 수행
- 다중 스레드 기반 백업 처리
- 간단한 CLI 인터페이스
